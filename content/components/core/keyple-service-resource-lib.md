## Overview

|                                                            |                                                                                                                      |
|------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| {{< icon name="file-alt" pack="fas" >}} **Description**    | The **Keyple Service Resource Library** is an **add-on** providing profile-based card resource allocation mechanism. |
| {{< icon name="users" pack="fas" >}} **Intended Audience** | Developers creating Keyple-based applications or card extension add-ons.                                             |
| {{< icon name="laptop" pack="fas" >}} **Compatible OS**    | Windows, Linux, macOS, Android.                                                                                      |
