Keyple Service Java Lib (2.1.2) is available!

Calypso Networks Association
Last updated on 2023-01-11 1 min read Release

Published on January 10, 2023, this version implements the Calypsonet Terminal Reader Java API (1.2.0) and therefore provide a mean to retrieve the physical name of the protocol currently used by the card reader to communicate with the card.

For more information, please visit the component page.
