Detailed Description

Class containing the Set of keyple::core::seproxy::message::SeRequest used to make a default selection at the keyple::core::seproxy::event::ObservableReader level.
It implements the keyple::core::seproxy::event::AbstractDefaultSelectionsRequest class by providing it with a public constructor.
